# preprocesamiento-ciencia-datos
## Objetivo
--------
Proveer scripts y notebooks para el preprocesamiento de datos: limpieza, transformación, extracción de características y pipelines reproducibles.

## Estructura básica de carpetas
----------------------------
- data/
  - raw/         # datos originales (no versionar grandes binarios)
  - processed/   # datos preparados para modelado
- notebooks/     # análisis exploratorio y pruebas
- src/           # código fuente (módulos de preprocesamiento)
  - preprocessing/
- tests/         # pruebas unitarias
- docs/          # documentación
